# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-09

### Added

- Website compliance scanner with scoring (0-100) and 10 issue categories.
- AI-powered document generation for 15 document types via OpenAI API.
- JWT authentication (register, login, token refresh).
- Credit-based system for document generation.
- Stripe payment integration for credit purchases.
- PDF export for generated documents.
- React 18 frontend with Tailwind CSS.
- Docker multi-stage build with PostgreSQL support.
- Flyway database migrations for H2 and PostgreSQL.
- SPA routing with Spring Boot.
